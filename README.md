# 👑 Vistula Family - Онлайн Мафия

Веб-приложение для организации онлайн-игр в Мафию с системой ролей и синхронизацией в реальном времени.

## ✨ Особенности

- 👑 **Система ведущего** с защитой паролем
- 🎮 **Присоединение игроков** по коду комнаты
- 🎲 **Автоматическое распределение ролей** среди игроков
- 🔄 **Синхронизация в реальном времени** через Firebase
- 📱 **Адаптивный дизайн** для мобильных устройств
- 🌙 **Современный интерфейс** с темной темой
- 🎭 **Многоролевая система**: Мафия, Шериф, Доктор, Маньяк, Путана, Мирный
- 💾 **Локальное сохранение** заметок хоста в браузере

## 🚀 Быстрый старт

### Требования

- Node.js 16+ 
- npm или yarn

### Установка локально

```bash
# Установка зависимостей
npm install

# Копируем .env.example в .env.local и заполняем значения
cp .env.example .env.local

# Запуск dev server
npm run dev

# Build для production
npm run build

# Preview production build
npm run preview
```

Приложение доступно по адресу: [http://localhost:5173](http://localhost:5173)

## 🔧 Конфигурация Firebase

Перед первым запуском необходимо настроить Firebase. Следуйте инструкциям в файле [FIREBASE_SETUP.md](./FIREBASE_SETUP.md).

### Переменные окружения

1. Скопируйте `.env.example` в `.env.local`
2. Заполните Firebase конфиг значения
3. Файл `.env.local` **не коммитится в git** (находится в `.gitignore`)

## 🚀 Деплой на Cloudflare Pages

Подробная инструкция в [CLOUDFLARE_DEPLOYMENT.md](./CLOUDFLARE_DEPLOYMENT.md)

**Короче:**
1. Push код на GitHub
2. В Cloudflare Pages добавьте переменные окружения (VITE_*)

## 🎮 Как использовать

### Для Ведущего

1. Нажмите кнопку "👑 Ведущий" на главной странице
2. Введите пароль: `7250mafia!`
3. Выберите роли и их количество
4. Поделитесь кодом комнаты с игроками
5. Нажмите "▶️ Начать" когда все игроки присоединились

### Для Игрока

1. Нажмите кнопку "🎮 Играть" на главной странице
2. Введите код комнаты и своё имя
3. Ожидайте, пока ведущий запустит игру
4. Получите свою роль при старте игры

## 📖 Роли в игре

- **🔪 Мафия** - Убивает мирных по ночам
- **⭐ Шериф** - Раскрывает мафию по дню
- **💊 Доктор** - Спасает игрока по ночам
- **😈 Маньяк** - Убивает каждую ночь без ограничений
- **👯 Путана** - Блокирует способность другого игрока по ночам
- **👤 Мирный** - Голосует в дневное время

## 🔐 Security Notes

Текущая конфигурация предназначена для локальной разработки. Для production используйте:
- Более строгие Firebase Security Rules
- Аутентификацию
- HTTPS
- Rate limiting
- Проверку входных данных на сервере
